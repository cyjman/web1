HTML 이란 무엇인가?

Hypertext Markup Language(HTML) is the standard mark up langauge for creating web pagesand web applications.

Web browsers receive HTML documents from a web server or from local storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the appearance of the document.

HTML elements are the bulding blocks of HTML pages. With HTML constructs, images and other objects, such as interactive forms, may be embedded into the rendered page.
